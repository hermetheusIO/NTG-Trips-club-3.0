# NonTourist Guide - Trips Club 3.0

A platform for authentic travel experiences in Portugal.

## About

NonTourist Guide offers curated trips and experiences that go beyond typical tourist destinations, 
providing authentic local experiences in Coimbra and surrounding regions.

## Tech Stack

- React 19 + TypeScript
- Express.js backend
- PostgreSQL with Drizzle ORM
- Tailwind CSS
- Vite

## Development

```bash
npm install
npm run dev
```
